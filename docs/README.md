## What is Flugins?

Flugins is a collection of Claude Code plugins that extend the functionality of Claude Code with custom tools and features. It provides a marketplace-based system for discovering, installing, and managing plugins.

## Quick Start

```bash
# Add the marketplace
/plugin marketplace add Flopsstuff/flugins

# Install a plugin
/plugin install <plugin-name>@flugins

# Update plugins
/plugin update
```

For more detailed information, see the [Getting Started](getting-started.md) guide.
